# ApricotPlayer 1.0 Roadmap

This beta cycle should keep the stable `main` branch untouched until the full 1.0 set is ready.

Planned feature order:

1. [x] Diagnostic report for testers.
2. [x] Transcript and captions browser with search, copy, and timestamp jump.
3. [x] Bookmarks and named markers.
4. [x] Audio normalization / ReplayGain-style playback mode.
5. [x] Equalizer Profiles 2.0: rename, delete, import, export, per-device defaults, and A/B compare. Beta 13 adds per-output-device default presets.
6. [x] Command search / action finder with a shortcut that does not use Ctrl+Shift+P.
7. [ ] Better queue and session restore. Beta 14 adds a manual Resume last playback session entry; deeper queue restore remains planned.
8. [x] Comments 2.0: sorting, search, replies, copy, and author/channel actions.
9. [ ] Podcast mode: played state, downloads, resume, chapters, transcripts, and per-podcast speed presets. Beta 15 adds played/unplayed state with a shortcut and refresh preservation. Beta 16 adds episode resume-progress labels and progress clearing.
10. [ ] Stable/beta release-channel polish. Beta 12 adds optional shortcut labels for beta discoverability; final 1.0 channel polish remains planned.
