# ApricotPlayer 1.0.0 Beta 17

This beta continues the podcast roadmap with per-podcast playback speed presets.

## New
- RSS/podcast feeds can now remember their own playback speed.
- Feed context menu: choose a podcast speed preset, or choose global speed to clear the preset.
- Player shortcut: `Ctrl+Shift+E` saves the current playback speed as the preset for the active podcast.
- The same action is also available from the player button row, player context menu, and Action finder while a podcast episode is active.

## Notes
- Podcast speed presets are preserved when feeds refresh.
- Normal YouTube, local file, and non-podcast playback still use the existing global speed path.
- This build does not change YouTube resolving, EQ, bass boost, volume boost, or seek timing.
