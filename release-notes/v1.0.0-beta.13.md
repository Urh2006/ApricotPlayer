# ApricotPlayer 1.0.0 Beta 13

This beta completes the Equalizer Profiles 2.0 roadmap item by adding per-output-device default presets.

## New
- Added per-output-device equalizer presets. When global EQ is enabled, Apricot can now use one preset for speakers, another for headphones, and another for `auto`.
- Added a Settings control in the Equalizer section for choosing the EQ preset tied to the current default output device.
- Added two Player Equalizer buttons: Save as default for this output device and Clear output device default.
- Diagnostic reports now show the effective EQ preset and the number of saved output-device EQ mappings.

## Notes
- This does not change EQ band math, slider behavior, bass boost, clipping protection, mpv startup, or stream resolving.
- Device defaults only choose which existing EQ preset is used. The actual filter chain stays the same as beta 12.
- Stable users remain on the stable channel unless they intentionally switch to beta updates.
