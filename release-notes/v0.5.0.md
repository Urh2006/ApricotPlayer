# What's new in version 0.5.0

- Added a History screen for recently played and downloaded items.
- Added a Subscriptions screen for YouTube channels.
- Added `Ctrl+Shift+S` to subscribe to the focused item's channel from search results, favorites, history, or the player.
- Added context menu subscription actions so subscription features are available without memorizing shortcuts.
- Added manual and automatic subscription checks for new channel videos.
- Added Windows notifications for new videos from subscribed channels.
- Added settings for subscription checking, check interval, notifications, history limit, and optional system tray close behavior.
- Added optional system tray mode: the close button or Alt+F4 can hide ApricotPlayer, while the Exit command closes it completely.
- Renamed the portable release package to `ApricotPlayer.zip` and kept compatibility with the old portable updater asset name.

## Downloads

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayer.zip` for portable builds.
