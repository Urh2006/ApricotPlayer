# ApricotPlayer 1.0.0 Beta 7

This prerelease is a beta-channel hotfix for startup audio bursts.

## Fixes
- Fixed the remaining loud/crackling burst that could happen when a local file starts playing audio immediately at 0.0 or 0.1 seconds.
- Apricot now starts mpv briefly muted and paused, applies the intended volume and volume limit, then immediately releases playback. This prevents the first audio frame from escaping before the player state is ready.
- If Start paused is enabled, Apricot still prepares the audio state but leaves playback paused.

## Notes
- This is intentionally a small audio startup fix. No unrelated volume or EQ behavior was changed.
