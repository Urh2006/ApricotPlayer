## What's new?

- updater now shows a visible progress dialog as soon as you choose `Update now`
- updater failures now appear as an error dialog instead of silently sitting in the status bar
- updater now uses GitHub's public release download URL first, with an API fallback for private releases
- playback speed announcements always use two decimals: `0.25x`, `0.50x`, `1.10x`, and so on
- speed control now includes `0.25x`

## Note

This release is intended to make updating test builds much easier, especially after switching the repository to public visibility.
