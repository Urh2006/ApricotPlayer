## What's new?

- Fixed a packaged-player crash where opening a video could fail with `NoneType` stream write errors.
- Queued downloads now have their own `Queued videos for download` screen.
- The main menu now opens the queued-download screen when items are waiting instead of running `Download all` immediately.
- In the queued-download screen, use Enter to download the selected item with its queued format.
- The queued-download screen also supports the context menu, `Ctrl+Shift+A`, `Ctrl+Shift+D`, and a tab-reachable `Download all` button.
- Batch downloads now clear the queue immediately, so the queued option disappears when nothing is waiting.
- Batch downloads now announce when they start and when each queued item begins downloading.
