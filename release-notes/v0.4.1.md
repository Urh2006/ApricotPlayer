# What's new in version 0.4.1

- Added ten more UI languages: German, French, Spanish, Portuguese, Italian, Polish, Dutch, Swedish, Croatian, and Serbian.
- Fixed remaining untranslated Slovenian labels and status messages.
- Language selection now uses the shared language registry, so future languages and translations stay consistent.
- Pitch-control mode choices are now localized in Settings while keeping the stored settings values stable.
- README now documents the localization rule: every new UI string must be translated for all supported languages in the same release.

## Downloads

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayerPortable.zip` for portable builds.
