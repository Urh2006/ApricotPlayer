# What's new in version 0.4.3

- Video downloads now default to MP4 instead of the previous best-available format that could produce WebM files.
- Settings now has friendly video download format choices: MP4 recommended, best available quality, MP4 single file, and smallest file.
- Existing installs using the old default `bestvideo+bestaudio/best` setting are migrated automatically to MP4 recommended.
- MP4 modes ask yt-dlp/ffmpeg to merge output as `.mp4`.
- The new labels are translated across all supported languages.

## Downloads

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayerPortable.zip` for portable builds.
