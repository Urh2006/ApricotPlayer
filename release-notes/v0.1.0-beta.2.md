## What's new

- updater now shows the changelog before the update prompt
- release publishing now requires notes, so each version has a proper "what's new" entry
- added player pitch control with `Ctrl+Up` and `Ctrl+Down`
- `S` and `D` continue to control playback speed in player mode

## Notes

- automatic app updates still require access to the private GitHub repository
- testers can authenticate either with GitHub CLI or by entering a GitHub token in settings
