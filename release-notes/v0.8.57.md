# What's new in version 0.8.57

- Added an optional clipping-protection setting for Volume boost + EQ/Bass boost. It uses a soft limiter only, so EQ band gains stay independent and ApricotPlayer does not bring back the automatic global headroom layer from older builds.
- Fixed Channel options > Popular videos so it ranks the channel by all-time view counts instead of sorting only the latest loaded batch.
- Popular videos now keeps ApricotPlayer's dynamic loading behavior: dynamic mode shows the first 20 top videos and reveals more in 20-result pages; fixed result limits show that many top videos.
