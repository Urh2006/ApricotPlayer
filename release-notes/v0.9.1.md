# What's new in version 0.9.1

- YouTube comments now default to yt-dlp when no YouTube Data API key is configured, so comments can still load without a key.
- If a YouTube Data API key is configured, ApricotPlayer still uses it first. If that first API request fails, ApricotPlayer now falls back to yt-dlp instead of stopping immediately.
- Settings, Cookies and network now includes an Obtain YouTube API key button that opens the Google Cloud credentials page.
- Comment loading now uses a stable YouTube watch URL, avoiding direct stream URLs from player metadata.
- Lyrics and chapter loading are more stable when you move quickly between player items.
