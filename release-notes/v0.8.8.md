# What's new in version 0.8.8

- Fixed background playback navigation so the player stays alive when you browse away with background playback enabled, while Escape from the full player still stops playback and returns to the previous results.
- Added player context-menu actions for download audio/video, favorites, playlists, playback queue, browser, copy URL, copy stream URL, output devices, equalizer, and close player.
- Added configurable player shortcuts: `F3` bass boost, `F4` equalizer, `R` repeat, and `Shift+S` shuffle.
- Added whole-folder local playback from the main menu with dynamic browsing and shuffle support.
- Preserved session volume and player EQ/bass/boost state when moving to the next track.
- Raised dynamic search/channel/playlist loading past the old 250-result limit.
- Fixed folder conversion so failed files are counted and skipped instead of stopping the whole folder.
- Improved Chrome-family cookie diagnostics for DPAPI/decode failures and kept the DevTools fallback for difficult profiles.
- Expanded Trending country choices for testers who enable Trending in Settings.
