# What's new in version 0.9.3

- Podcast chapters now work from more sources. ApricotPlayer still uses embedded chapters when mpv exposes them, and now also reads inline Podlove/PSC chapters from podcast RSS items.
- Podcasting 2.0 `podcast:chapters` JSON URLs are supported and loaded only when you open Chapters or use chapter navigation.
- Chapter time parsing now handles `00:01:23.500`, `01:23`, numeric seconds, `start`, and `startTime` values.
- Settings now has a Check yt-dlp updates now button next to the automatic yt-dlp update setting.
- Manual yt-dlp checks announce whether YouTube support was updated or is already up to date.
