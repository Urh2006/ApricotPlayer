# ApricotPlayer 0.9.51

This release fixes one hidden regression found during the 0.9.50 review.

## Fixes

- Dynamic result loading now uses the same yt-dlp cookie retry and YouTube auth hint behaviour as the first search page. This covers loading the next 20 results, channel videos, playlists, and other collection pages.

## Review Notes

- Compared current 0.9.50 against the old `07fc4c4` v0.9.18 commit. The current player path is not obviously slower than the old monolith: the stream resolver now prefers audio-only HTTPS streams, the mpv cache settings are still present, and the stream URL cache is slimmed so it no longer keeps huge yt-dlp fields like formats, captions, comments, thumbnails, or heatmaps.
