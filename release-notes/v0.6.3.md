# What's new in version 0.6.3

- Duplicate shortcut protection: Settings now warns when a captured keyboard shortcut is already assigned to another action, so users cannot accidentally give two actions the same shortcut.
- New public default download layout: fresh installs use `Downloads\ApricotPlayer`, with YouTube audio/video downloads under `music` and podcast episode downloads under `podcasts`.
- Channel and playlist downloads now create their own folders inside the music download area.
- Podcast episodes and complete podcast/RSS feeds now download into podcast-named folders inside the podcasts area.
- Added "Download entire feed" actions for podcast/RSS feed lists and episode lists.
- Podcast episodes can be queued for audio download with Shift+A and then downloaded through the existing queue flow.
- The default New subscription videos shortcut is now `N`, keeping the public default shortcut set unique.
