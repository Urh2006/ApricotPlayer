# What's new in version 0.4.4

- Fixed the update dialog buttons so `Update now` and `Skip this version` explicitly trigger the correct action.
- `Skip this version` now announces that the version was skipped instead of only changing the status bar.
- Updater logging now starts at the update prompt, so `%APPDATA%\ApricotPlayer\updater.log` records prompt display, button choice, download start, downloaded package size, and updater script launch.
- Verified the prompt in an automated wx test: skip returns skip and update returns update.

## Downloads

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayerPortable.zip` for portable builds.
