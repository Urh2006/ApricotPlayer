# ApricotPlayer 0.9.26

## Bug Fixes and Stability Improvements

- **Critical Auto-Update Fix**: Fixed a crash during the auto-update process where clicking "Update" would fail with `Could not check app updates: name 'MainFrame' is not defined` due to modularization dependencies.
- **Codebase Stabilization**: Completely decoupled core utility and updater modules from `MainFrame` references by using local static calls and class method bindings, eliminating any remaining reference/import lookup errors.
- **Data Safety**: Verified that updater settings, user playlists, subscriptions, history, and preferences remain fully intact and are securely preserved when updating.

*Note: All core functionality has been verified for stability. Entry-point script (wx_main.py) is kept under 1,000 lines, ensuring clean modularity and stable operation.*
