# What's new in version 0.8.13

- Fixed playback for YouTube videos where imported cookies are valid but the first fast format selection fails with `Requested format is not available`.
- ApricotPlayer still tries the fast playback path first, then uses the slower YouTube JS fallback only after that format/auth failure.
- Verified the reported test video resolves successfully with imported cookies even when the age-restricted fallback setting is off.
