# ApricotPlayer 0.9.56

Hotfix release for player Tab and Shift+Tab responsiveness.

## Fixes
- Reduced the delay when tabbing through the player, embedded result list, and player buttons.
- Apricot now lets wx handle normal button/list/checkbox focus movement natively again.
- The custom Tab fallback remains only for the embedded mpv player panel, where native focus movement can still get stuck.
- Kept the 0.9.55 fix where background-player controls no longer reverse or disrupt search/channel/playlist screen tab order.
