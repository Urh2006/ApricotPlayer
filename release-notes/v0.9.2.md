# What's new in version 0.9.2

- Opening a new local file from disk while ApricotPlayer is already running now updates the player first, then brings ApricotPlayer forward. This prevents screen readers from announcing the previous file title first.
- Added Copy link at current time in the player for YouTube videos. The default shortcut is Ctrl+Shift+L, and the player context menu shows the shortcut.
- Lyrics now defaults to Ctrl+Shift+Y to avoid conflicting with Copy link at current time.
- Local files now say Copy path instead of Copy link in the player, and Copy direct media URL is hidden for local files.
- Show video details is now simply Show details, including the shortcut name in settings.
