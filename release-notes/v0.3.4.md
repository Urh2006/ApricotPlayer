## What's new in 0.3.4

- Optimized the Windows build so ApricotPlayer opens faster.
- Removed the broad `--collect-all wx` packaging step, which was bundling many unused wx demo/helper modules.
- Excluded unused heavyweight modules such as IPython, matplotlib, numpy, PIL, tkinter, and jedi from the packaged app.
- The installed executable is smaller, so the one-file app has less to unpack before the window appears.
- `yt-dlp` is now loaded only when a YouTube action needs it, which helps the main menu appear sooner.
- Automatic update checks still run when enabled, but now start a few seconds after the main menu appears.
