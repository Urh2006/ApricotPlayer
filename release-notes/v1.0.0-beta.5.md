# ApricotPlayer 1.0.0 Beta 5

This prerelease continues the 1.0 beta channel. Stable-channel users should stay on the latest 0.9.x release unless they intentionally switch to beta.

## New
- Added `Ctrl+Shift+PageDown` for Player: play related video. This plays a related/suggested video from the currently playing YouTube item.
- Kept `Ctrl+PageDown` as the normal next command for results, folders, playlists, and the playback queue.
- Added Equalizer profile import and export as JSON files.
- Added an A/B compare button in the player Equalizer dialog so testers can switch between the original EQ state and the edited preview.

## Fixes
- Fixed Autoplay related suggested videos so the setting works after a YouTube video finishes, even when normal autoplay-next is not enabled.
- Removed the accidental plain F1 marked-preview shortcut. Marked preview remains on the configured shortcut, defaulting to `P`.

## Notes
- The reported local-file volume jump was reviewed but not changed in this build because there was no clear reproducible code fault yet. Please send a diagnostic report or exact steps if it still happens.
