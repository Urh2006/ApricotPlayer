# What's new in version 0.6.10.2

- Fixed the ApricotPlayer system tray icon so Applications key and Shift+F10 open the tray context menu instead of immediately restoring the main window.
- Fixed Keyboard shortcuts settings so pressing Space on the Save button no longer gets treated as a duplicate play/pause shortcut.
- Moved the recommended pitch and speed processing choices to the top of their Settings combo boxes: MPV for pitch and Rubberband for speed.
