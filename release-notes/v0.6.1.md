# What's new in version 0.6.1

- Added podcast search using the Apple Podcasts directory through the iTunes Search API.
- Direct RSS and Atom feed URLs are still supported.
- Added a Podcasts and RSS settings section with provider, country, search result limit, maximum episodes per feed, startup refresh, automatic refresh, and refresh interval options.
- Added settings to hide Podcasts and RSS feeds or History from the main menu.
- Removed Choose download folder from the main menu because the same control is already in Settings.
- When History is disabled, new played and downloaded items are no longer recorded.

## Downloads

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayer.zip` for portable builds.
