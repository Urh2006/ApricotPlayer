# What's new in version 0.5.1

- Fixed the search result context menu for channels and playlists.
- Channels and playlists no longer show single-video actions like Play, Download audio, or Download video in the result context menu.
- Channel and playlist context menus now show collection-aware actions such as Open channel videos, Open playlist videos, Download channel, or Download playlist.
- Added a New videos action in Subscriptions that opens only videos found since the last subscription check.
- Pressing Enter on a subscribed channel opens all channel videos, and Escape returns to the Subscriptions list.
- Tightened library handling so saved channel and playlist items open as collections instead of single videos.

## Downloads

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayer.zip` for portable builds.
