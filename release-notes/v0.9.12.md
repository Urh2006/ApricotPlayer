# What's new in version 0.9.12

- Fixed session volume sometimes resetting between songs to 100, 0, or the saved default volume.
- Made ApricotPlayer carry the session volume itself instead of trusting unstable mpv volume reads during track changes.
- Prevented old pending volume key presses from affecting the next mpv process.
- Kept boosted session volumes above 100 when moving to previous or next items in the same player session.
