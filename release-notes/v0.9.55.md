# ApricotPlayer 0.9.55

Hotfix release for playback startup and background UI navigation.

## Fixes
- Removed the YouTube audio-only fallback from the fast playback path. YouTube videos now use progressive audio+video streams only, so starting playback and seeking immediately should feel like the older fast behaviour again.
- Bumped the stream URL cache profile so old cached m4a/audio-only URLs are ignored after updating.
- Kept SoundCloud and other non-YouTube audio-only sources working through a separate non-YouTube stream format.
- Fixed the background player section changing the tab order in search, channel, and playlist screens while playback continues in the background.
