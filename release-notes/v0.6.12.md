# What's new in version 0.6.12

- Restored fast playback startup by keeping the first yt-dlp playback/download attempt cookie-free.
- ApricotPlayer now retries with cached `cookies.txt` only after YouTube reports sign-in or bot-confirmation errors.
- The automatic browser profile export/repair flow now runs only as a second fallback when cached cookies are missing or stale.
- Normal downloads no longer inject cookies before an error, so they stay on the same fast path as playback.
