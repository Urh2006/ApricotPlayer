## What's new?

- new installs now default to English
- settings are saved in `%APPDATA%\ApricotPlayer\settings.json`
- settings now include `Restore to defaults`
- screen reader announcements now use NVDA directly when available, including `T`, `V`, `L`, speed, and pitch
- video details now use a standard read-only text field for better reading, arrow navigation, and copying
- speed and pitch controls are more sensitive, including `1.1x`
- returning to default `1.0x` speed or pitch plays a short confirmation sound
- dynamic result loading now fetches 20 results first and loads more only when you reach the end
- context menu download actions now show `Ctrl+Shift+A` and `Ctrl+Shift+D`

## Note

This release intentionally does not overwrite the user's existing `0.1.1` executable in Downloads during development, so the auto-updater can be tested from that build.
