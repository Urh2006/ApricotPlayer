# ApricotPlayer 1.0.0 Beta 11

This beta adds Comments 2.0 without touching the player startup, volume, bass boost, equalizer, or mpv audio handoff paths.

## New
- Added a search box to the YouTube Comments screen.
- Added comment sorting for original order, newest first, oldest first, most liked, and most replies.
- Added Copy comment and Copy visible comments.
- Added Open author channel when YouTube or yt-dlp exposes the comment author's channel URL.
- Comment replies are now included in comment search and copied comment text.

## Notes
- Sorting is local over the comments already loaded into the dialog. Load more still uses the existing YouTube API / yt-dlp loading path.
- Stable users remain on the stable channel unless they intentionally switch to beta updates.
