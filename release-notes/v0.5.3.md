# What's new in version 0.5.3

- Added a Notifications settings section with a master Windows notifications switch.
- Added separate toggles for new subscription video notifications and completed download notifications.
- Completed audio, video, batch, playlist, and channel downloads now send a Windows notification when ApricotPlayer is not focused.
- When ApricotPlayer is focused, completed downloads still use the existing popup or screen reader announcement setting.
- Keyboard shortcut fields now capture the shortcut when you focus the field and press the desired key combination.
- Tab and Shift+Tab still move between shortcut fields instead of being captured as shortcuts.
- Translated the new notification and shortcut capture strings across the supported language packs.

## Downloads

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayer.zip` for portable builds.
