# ApricotPlayer 1.0.0 Beta 2

Beta 2 continues the 1.0 feature cycle on the beta channel. Stable users stay on the latest stable release unless they switch to beta updates.

## Added

- Added a player transcript/captions browser.
- Added `Ctrl+Shift+T` for opening transcripts from the player.
- Added transcript search, Enter-to-jump timestamp playback, copy selected transcript line, copy full transcript, and copy timestamp link.
- Added local sidecar transcript support for `.vtt` and `.srt` files next to local media.
- Added a missing `Copy lyrics` button to the lyrics window.

## Notes

- Transcript loading prefers manual subtitles, then automatic captions, using the subtitle languages configured in settings.
- This is a beta prerelease and is published only on the beta channel.
