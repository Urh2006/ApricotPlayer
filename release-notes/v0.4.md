## What's new in 0.4

- Added `All` to YouTube search so videos, playlists, and channels can appear together.
- Playlist and channel results now have a context-menu download action:
  - `Download playlist`
  - `Download channel`
- Playlist/channel downloads create a dedicated folder under your configured download folder and put the videos inside it.
- Releases now publish:
  - `ApricotPlayerSetup.exe`
  - `ApricotPlayerPortable.zip`
- The raw single-file `ApricotPlayer.exe` is no longer published by default.
- Portable builds can update from the ZIP release by extracting the new version into the current app folder.
- Restored linked pitch/speed mode as `Linked pitch and speed - pitch keys change both`.
- In linked pitch/speed mode, pitch keys change pitch and speed together, but speed keys still change only speed.
