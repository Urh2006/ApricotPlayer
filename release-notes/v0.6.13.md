# What's new in version 0.6.13

- `Ctrl+Shift+V` now opens the Notification center directly from anywhere in ApricotPlayer.
- The keyboard shortcut settings label now says Notification center instead of New subscription videos.
- In the subscriptions screen, the same shortcut now opens the Notification center instead of only one selected channel's saved new videos.
- The Notification center preserves the selected item when the list refreshes.
