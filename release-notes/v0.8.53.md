# What's new in version 0.8.53

- The whole 10-band equalizer path was checked and polished, not only the low bands.
- Every EQ band now has an explicit filter width from 31 Hz through 16 kHz, so each slider targets a stable audio range.
- 16 kHz now uses a high-shelf treble filter, making the top band more audible and consistent.
- EQ sliders now listen to both normal slider and wx scroll events, so keyboard, page, arrow, mouse, and assistive-control changes refresh the same live preview path.
- mpv equalizer filter replacement now retries when the player is briefly busy, reducing cases where a slider moves but the active audio filter does not update.
