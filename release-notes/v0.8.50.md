## What's new in version 0.8.50

- Playlist and channel context menus now use a Download submenu with Audio and Video choices. The keyboard shortcuts still work directly: `Ctrl+Shift+A` for audio and `Ctrl+Shift+D` for video.
- Empty playlists and favorites now hide actions that cannot be used yet.
- The player Fullscreen checkbox now toggles reliably with Enter.
- Equalizer sliders now use correct keyboard movement across the full 24 dB range, including 3 dB PageUp/PageDown steps.
- Bass boost now layers on top of the current player or global equalizer instead of resetting equalizer settings.
- Background Next/Previous playback now keeps using the embedded player panel and preserves focus instead of opening a separate mpv window.
- Old custom F5 shortcut assignments are repaired back to their default action bindings.
