# ApricotPlayer 1.0.0 Beta 3

Beta 3 continues the 1.0 beta cycle on the beta channel. Stable users stay on the latest stable release unless they switch to beta updates.

## Added

- Added persistent bookmarks and named markers.
- Added a Bookmarks item to the main menu.
- Added `Ctrl+Alt+K` to open all bookmarks.
- Added `Ctrl+Shift+B` to add a bookmark at the current player position.
- Added `Ctrl+Shift+K` to open bookmarks for the current player item.
- Added player buttons and context menu entries for Add bookmark and Bookmarks.
- Added bookmark rename, delete, play, and copy timestamp link actions.
- Added bookmark start-position playback so opening a bookmark starts at the saved timestamp instead of the normal resume position.

## Notes

- Bookmarks are saved in ApricotPlayer's app data as `bookmarks.json`.
- Timestamp links are available for YouTube bookmarks; local file bookmarks still play and seek normally.
- This is a beta prerelease and is published only on the beta channel.
