# What's new in version 0.8.10

- Dynamic result mode no longer has ApricotPlayer's old app-side cap. Searches, channels, playlists, and local folders keep loading 20 more items until the source stops returning more.
- Previously loaded dynamic results stay in the list, so you can keep arrowing back through everything already fetched.
- Added an Add to playlist button to the full player and background Player section for the currently playing YouTube item, podcast episode, or local file.
- Added Add to playlist to podcast episode context menus and made player Add to playlist prefer the currently playing item instead of unrelated queued downloads.
- Playlist downloads now skip local files instead of trying to download already-local media through `yt-dlp`.
- Rechecked folder conversion so one failed file is counted and skipped without stopping the whole folder conversion.
