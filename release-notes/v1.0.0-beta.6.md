# ApricotPlayer 1.0.0 Beta 6

This prerelease continues the 1.0 beta channel. Stable-channel users should stay on the latest 0.9.x release unless they intentionally switch to beta.

## New
- Added Action finder. Press `Ctrl+Shift+J` or open it from the main menu, type part of an action name, and press Enter to run it.
- Action finder includes global actions such as Search, Play folder, Settings, History, Podcasts/RSS, Queue, and Diagnostics.
- When playback is active, Action finder also includes player actions such as Play/Pause, Previous, Next, Copy link/path, Details, Output devices, Equalizer, Audio normalization, Bookmarks, Chapters, Transcript, Lyrics, Comments for YouTube, and Close player.

## Fixes
- Reduced the startup loudness jump with boosted EQ and clipping protection. Apricot now starts mpv with the active EQ/clipping filter already attached, then verifies it shortly after startup.

## Notes
- The diagnostic report showed default volume 20 but session/mpv volume 100. That can be valid if the user changed volume during the open player session. The concrete issue fixed here is the delayed EQ/clipping-protection application that can sound like a sudden volume rise followed by a correction.
