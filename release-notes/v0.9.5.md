# What's new in version 0.9.5

- File converter and Folder converter now expose one clearer format combo box named "Format to convert to" instead of the old separate "Detected format" field.
- Opening a new local file while ApricotPlayer is hidden in the system tray now brings the existing window forward immediately.
- The new file title is applied before the window is restored, so screen readers should not announce the previous file title first.
