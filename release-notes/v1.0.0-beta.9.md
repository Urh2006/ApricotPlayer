# ApricotPlayer 1.0.0 Beta 9

This beta keeps the beta 8 audio handoff protection, but removes the possible stop-time delay from that fix.

## Fixes
- Sends the old-player mute, volume-zero, and pause handoff in one best-effort mpv IPC write instead of three separate calls.
- Uses a no-wait pipe open for this handoff, so if mpv IPC is not immediately available Apricot skips the protection and continues stopping the player rather than delaying navigation/playback.

## Notes
- This is intentionally small. It does not change EQ, volume math, or normal playback startup.
- Local installs are not updated automatically from this build unless you choose the beta channel/update manually.
