# What's new in version 0.4.7

- Fixed the yt-dlp component updater announcement so screen readers only say "updating components" when a newer yt-dlp package was actually found and installation is starting.
- Quiet no-update component checks at startup, so ApricotPlayer does not announce component updating when there is nothing to update.
- Published fresh installer and portable ZIP release assets for testing updates from 0.4.6.

## Downloads

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayerPortable.zip` for portable builds.
