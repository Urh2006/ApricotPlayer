# ApricotPlayer 0.9.9

- Fixed results-list navigation after returning from the player: Up/Down/Page/Home/End now navigate results instead of changing player volume.
- Fixed Ctrl+PageDown at the end of currently loaded results: ApricotPlayer now loads the next dynamic page and continues playback instead of getting stuck on "No next item".
- Restored the correct dynamic-loading context when returning from player results, so stale playlist/channel URLs are not reused for unrelated result lists.
- Dynamic-load errors are now announced through status/screen reader feedback instead of opening a modal dialog while navigating results.
