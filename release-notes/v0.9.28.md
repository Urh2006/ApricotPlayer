# ApricotPlayer 0.9.28

## Critical Keyboard & Tab Navigation Hotfixes

- **Search Box Typing Fix**: Resolved a critical keyboard focus interception bug inside `wx.EVT_CHAR_HOOK` which swallowed typing keys and prevented any input inside the search query box. The handler now natively bypasses key interception for all editable text fields (`focus_accepts_text`), allowing seamless typing, backspacing, selecting, and arrow navigation.
- **Settings Tab Navigation Fix**: Bypassed custom Tab key hook interception outside active player/background player panels. Hitting Tab or Shift+Tab inside settings text fields or choices now flawlessly shifts focus natively between all settings controls.
- **Auto-Update Safety**: Kept the core entry-point script (`wx_main.py`) under 1,000 lines, ensuring clean modularity and robust operation.
