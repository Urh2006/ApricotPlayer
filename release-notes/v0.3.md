## What's new?

- Added a Windows installer: `ApricotPlayerSetup.exe`.
- The installer adds ApricotPlayer to the Start Menu and can create a desktop shortcut.
- User settings remain per-user in `%APPDATA%\ApricotPlayer\settings.json`.
- The updater now prefers installer releases when available, with the old portable `.exe` replacement kept as a fallback.
- The updater now checks GitHub's `/releases/latest` endpoint directly instead of relying on release list ordering.
- GitHub releases can now include both `ApricotPlayerSetup.exe` and the portable `ApricotPlayer.exe`.
- The public README no longer references local developer download paths.

## Migration note

Version `0.2.2` still updates through the portable `.exe` updater, so this release also includes `ApricotPlayer.exe`. After users are on `0.3`, future releases can update through the installer asset.
