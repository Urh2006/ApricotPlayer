# What's new in version 0.8.51

- Equalizer live preview is now debounced and retried, so rapid slider movement no longer leaves old or missing mpv filters behind.
- The 31 Hz, 62 Hz, and 125 Hz bands now use wider octave EQ bands so sub bass and low bass changes are clearly audible.
- Volume boost plus Bass boost now adds EQ headroom and a limiter to reduce crackling and clipping.
- Fast repeated Next/Previous playback requests now ignore stale stream-resolution results, preventing two items from starting at the same time.
- Custom equalizer preset renaming no longer redraws the Settings page on focus loss, so Tab and Escape navigation keep working.
- The default playback volume slider now stops at 100 unless Volume boost on by default is enabled; boosted defaults can still go up to 300.
- The Seek seconds setting now includes 0.1, 0.25, 0.5, 0.75, 1, 1.5, 2, 2.5, and more, and the small seek shortcuts use the configured value.
