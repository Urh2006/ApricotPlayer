# What's new in version 0.6.10.7

- Changed `Shift+A` into a general video selection shortcut for downloads or adding to playlists.
- Removed the separate `Shift+D` queue-video shortcut from Keyboard Shortcuts settings.
- Adding selected videos to a user playlist now clears those selections from Current downloads.
- Current downloads now offers `Download all as audio` and `Download all as video`.
- Pressing `Enter` on a selected queued item now asks whether to download it as audio or video when no format was chosen yet.
