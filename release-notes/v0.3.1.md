## What's new in 0.3.1

- Fixed `Ctrl+Shift+A` and `Ctrl+Shift+D` so audio/video downloads work from the results list and while playback is focused in the player panel.
- Added app-wide download accelerators for more reliable Windows keyboard handling.
- Improved the Chrome cookie database error with a useful hint: close Chrome completely, use Edge or Firefox, or choose an exported cookies.txt file.
- Pitch control now announces that pitch is unavailable when mpv is not ready instead of saying that timing is unavailable.
- Installer-based updates now request the desktop shortcut task so installed test builds stay easy to launch.
