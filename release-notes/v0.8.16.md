# What's new in version 0.8.16

- Fixed background playback so Escape returns one level back without stopping playback when background playback is enabled.
- The Player Play button now changes to Pause while media is playing, and back to Play when paused or finished.
- Renamed background controls to Open player screen and Close, and removed the extra background playback announcement.
- Rebuilt Play from folder: ApricotPlayer now scans the selected folder, shows an accessible file list, and lets you Play folder, Shuffle folder, or Add folder to queue.
- Added queue reordering from the playback queue with Move up and Move down in the context menu.
- Changed default player shortcuts: V announces the current volume, and F7 opens video details.
