# ApricotPlayer 1.0.0 Beta 1

First 1.0 beta release.

## New
- Added Copy diagnostic report for testers and support.
- The diagnostic report copies app version, update channel, yt-dlp/mpv/FFmpeg status, active player state, audio/session settings, current media metadata, queue/result counts, and redacted tails of mpv/updater logs to the clipboard.
- Added the diagnostic report to the main menu and keyboard shortcuts settings.
- Default shortcut: Ctrl+Alt+Shift+D.
- Added `docs/ROADMAP_1.0.md` so the agreed 1.0 beta feature list stays in the repository while the beta cycle continues.

## Notes
- This is a GitHub prerelease on the beta branch. The stable updater channel should ignore it; the beta channel should offer it.
