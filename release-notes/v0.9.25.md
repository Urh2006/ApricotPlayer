# ApricotPlayer 0.9.25

## Bug Fixes

- **Critical Keyboard Fix**: Fixed a severe issue introduced in 0.9.24 where keyboard inputs (arrows, letters, and other shortcuts) would completely fail to respond when the app was launched as a packaged executable.
- **Context Menu Cleanup**: Fixed an issue where YouTube-specific options (like subscribe, download, etc.) were incorrectly visible in the context menu when playing from a local folder.

*Note: Data persistence was confirmed. Updating from previous versions will securely preserve your settings.*
