## What's new

- player announcements now send explicit Windows accessibility events, so NVDA should speak them more reliably
- pressing `T` in player mode should announce elapsed, remaining, and total duration instead of only changing the status bar
- pressing `S` and `D` now moves through clean playback speed steps: `0.5x`, `0.75x`, `1.0x`, `1.25x`, `1.5x`, `1.75x`, `2.0x`, `2.5x`, `3.0x`, and `4.0x`
