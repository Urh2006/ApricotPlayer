# What's new in version 0.6.10

- Fixed updater version comparison so future hotfix versions like 0.6.10.1 are detected correctly.
- Used a three-part 0.6.10 release number so older 0.6.9-era updaters can detect this update.
- Release checks now select the highest public GitHub release version, even if GitHub's latest marker is stale.
- Hardened updater downloads with exact asset-name checks, trusted HTTPS source checks, file-size checks, SHA-256 verification when published, and safe ZIP path validation.
- Hardened yt-dlp component updates by verifying PyPI wheel SHA-256 digests and extracting wheels with path traversal protection.
- Optimized Settings section navigation so moving with arrow keys no longer rebuilds the settings pane on every key press.
- Rebuilt Keyboard shortcuts settings into a lightweight action list plus one shortcut capture field for smoother NVDA navigation.
