# What's new in version 0.5.2

- Verified the GitHub repository is public and simplified update checks to use the public ApricotPlayer release endpoints.
- Removed user-facing GitHub owner, repository, and private update token settings.
- Added an accessible Keyboard shortcuts settings section so users can customize the default shortcuts.
- NVDA announcements now send both speech and braille messages when the NVDA controller supports braille output.
- Status messages also update the status bar and raise accessibility events for better braille/display behavior.
- Empty lists now expose readable placeholder items such as No results, No favorites, No subscriptions, and No queued downloads.
- Settings now announces Settings saved after pressing Save.
- Added 15 language choices and translated the new shortcut, empty-list, and important status announcement strings.

## Downloads

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayer.zip` for portable builds.
