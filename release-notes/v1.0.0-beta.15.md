# ApricotPlayer 1.0.0 Beta 15

This beta continues the Podcast mode roadmap work with played/unplayed state for podcast and RSS episodes.

## New
- Added played markers to podcast/RSS episode lists.
- Added played counts to podcast/RSS feed list entries.
- Added `Ctrl+Shift+X` for marking the selected episode played or unplayed.
- Added the same played/unplayed toggle to the episode context menu.
- Podcast/RSS episodes now automatically mark as played when playback reaches the end.
- Feed refreshes preserve played state by episode URL, GUID, webpage URL, or title.

## Notes
- This beta does not change YouTube resolving, mpv startup, seek handling, EQ, bass boost, volume boost, or standard Next/Previous navigation.
- Stable users remain on the stable channel unless they intentionally switch to beta updates.
