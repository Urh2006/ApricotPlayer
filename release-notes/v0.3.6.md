## What's new in 0.3.6

- Rebuilt pitch control so playback speed and pitch stay independent.
- Removed the confusing legacy `linked speed` pitch mode. Existing settings using it are migrated to the recommended independent Rubberband mode.
- Pitch mode names are now more descriptive:
  - `Independent pitch - best quality (Rubberband)`
  - `Independent pitch - basic (mpv built-in)`
- Rubberband pitch changes now update ApricotPlayer's own labeled audio filter instead of replacing the full audio filter chain or changing speed.
