# What's new in version 0.4.6

- Fixed update checks on systems where Python/OpenSSL could not verify GitHub's certificate.
- GitHub and PyPI HTTPS requests now use the bundled `certifi` certificate store.
- ApricotPlayer now exits immediately after launching the updater helper, preventing the old app window from getting stuck as not responding.
- Updater helper scripts now wait up to 15 seconds for ApricotPlayer to close, then force-close only that ApricotPlayer process if needed.
- The installer and portable ZIP updater paths both include the same certificate and shutdown fixes.

## Downloads

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayerPortable.zip` for portable builds.
