Beta 4 continues the 1.0 beta channel. Stable users remain on stable releases unless they switch to beta updates.

## New
- Added a player Audio normalization control for ReplayGain / loudness normalization.
- Added `Ctrl+Shift+G` to cycle audio normalization between Off, Track, and Album.
- Added the audio normalization shortcut to Settings > Keyboard shortcuts.

## Fixes
- Transcript loading now uses yt-dlp/YouTube-style request headers and falls back to yt-dlp subtitle download when direct caption loading fails.
- YouTube caption rate limits now show a friendly message instead of the raw `HTTP Error 429: Too Many Requests`.
- Plain F5 no longer announces duration; the player time announcement remains on `T` by default.
