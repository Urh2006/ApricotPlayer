# ApricotPlayer 0.9.7

- Restored queue priority for Next: if you manually add items to Playback queue, Next plays the queued item before falling back to the normal folder/result next item.
- Restored the same queue priority for automatic next playback when "Automatically play next item" is enabled.
- Kept the Play folder fix from 0.9.6 by ignoring old auto-generated folder queue entries instead of treating them as real user queue items.
- Updated next-item prefetch so it ignores legacy auto-folder queue entries and looks at the first real queued item.
