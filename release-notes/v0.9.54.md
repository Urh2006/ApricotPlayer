# ApricotPlayer 0.9.54

This release focuses on keyboard feel and result-menu stability.

## Fixes
- Restored hold-to-seek scrubbing for the player seek shortcuts. Holding seek forward or seek backward now keeps moving continuously instead of only working as separate repeated presses.
- Reduced UI blocking during held seeking by using short non-blocking mpv seek sends for the repeat path.
- Made player Tab and Shift+Tab routing more deterministic across embedded results, the player panel, navigation buttons, and player action buttons.
- Fixed playlist/channel context menus reading a stale result selection. Play playlist and Shuffle playlist now come from the actually selected result row.
