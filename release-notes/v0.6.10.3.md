# What's new in version 0.6.10.3

- Changed the default New subscription videos shortcut to `Ctrl+Shift+V`.
- Added startup repair for older settings files where New subscription videos was still assigned to `Space`, conflicting with player play/pause.
- Reworked the Audio quality setting so the combo box shows clear choices such as Best variable quality (VBR 0), 320 kbps, 256 kbps, and other VBR or bitrate options.
- Made normal app exit cleaner by stopping timers, tray resources, and the embedded player before the main window is destroyed.
