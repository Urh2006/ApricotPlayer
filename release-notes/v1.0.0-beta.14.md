# ApricotPlayer 1.0.0 Beta 14

This beta starts the queue/session restore roadmap work with a small, safe restore point for the last played item.

## New
- Added Resume last playback session to the main menu when a saved playback item exists.
- Added the same action to Action finder, so `Ctrl+Shift+J` can find it from anywhere.
- The restore action reuses Apricot's existing Resume where you left off setting for the actual start position.
- Diagnostic reports now include the saved last-session title.

## Notes
- This beta intentionally does not change YouTube resolving, mpv startup, seek handling, EQ, bass boost, volume boost, or Next/Previous navigation.
- Stable users remain on the stable channel unless they intentionally switch to beta updates.
