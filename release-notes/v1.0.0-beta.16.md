# ApricotPlayer 1.0.0 Beta 16

This beta continues the 1.0 roadmap with full-screen shortcut support and podcast resume polish.

## New
- Added `F11` as the default Player full-screen shortcut.
- Added Player full-screen to Settings > Keyboard shortcuts.
- Full-screen controls now show the configured shortcut when shortcut labels are enabled, including the background-player full-screen button.
- Podcast/RSS episode lists now show `resume at ...` when an unfinished episode has saved playback progress.
- Added `Ctrl+Shift+R` and a context-menu action to clear the selected podcast/RSS episode's saved progress.
- Marking a podcast/RSS episode played now also clears its saved resume progress.

## Notes
- This beta does not change YouTube resolving, mpv startup, seek handling, EQ, bass boost, volume boost, or standard Next/Previous navigation.
- Stable users remain on the stable channel unless they intentionally switch to beta updates.
