## What's new in 0.3.7

- Fixed the Rubberband runtime pitch update so repeated pitch changes update ApricotPlayer's existing mpv audio filter.
- This keeps pitch and playback speed independent while making `Ctrl+Up` and `Ctrl+Down` smoother.
- Includes the rebuilt independent pitch control from 0.3.6:
  - `Independent pitch - best quality (Rubberband)`
  - `Independent pitch - basic (mpv built-in)`
