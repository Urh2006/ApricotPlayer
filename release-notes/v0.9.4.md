# What's new in version 0.9.4

- Chapters now play with Enter: when focus is on a chapter, Enter jumps to that chapter and closes the Chapters dialog.
- Reopening ApricotPlayer from the desktop shortcut, system tray, or a local media file now restores the existing hidden window properly after Alt+F4-to-tray background playback.
- Play folder is isolated from previous YouTube results, so starting a folder while a YouTube item is playing no longer reuses the old result list.
- Folder playback queue state is more consistent for Play folder, individual local files, Previous, and Next.
- Local folder results now use natural numeric sorting, so `Telemach.mp3`, `Telemach(1).mp3`, `Telemach(2).mp3`, and `Telemach(10).mp3` appear in the expected order.
