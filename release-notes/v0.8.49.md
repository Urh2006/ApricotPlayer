## What's new in version 0.8.49

- Fixed dynamic result playback so Enter keeps the exact selected item even while more results are loading.
- Stabilized equalizer sliders and added the 6/12/18/24 dB range selector to the player equalizer.
- Fixed the player Fullscreen checkbox so Space and Enter toggle fullscreen correctly.
- Made Bass boost session-based, so it stays enabled for the next item until turned off.
- Added longer stream URL cache choices, including 7 days and permanent-until-YouTube-expires.
- Kept focus in the current list when using Next/Previous from results during background playback.
