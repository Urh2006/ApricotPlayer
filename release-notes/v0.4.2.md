# What's new in version 0.4.2

- Pitch changes are now retry-aware, so pressing `Ctrl+Up` or `Ctrl+Down` while mpv is still starting or briefly busy should no longer produce the transient `Pitch control is not available yet` message.
- mpv JSON IPC commands are now serialized through one lock, which prevents pitch, speed, time, and volume commands from colliding with each other.
- Rubberband pitch filter changes now verify mpv's response and retry cleanly when the audio filter is not ready yet.

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayerPortable.zip` for portable builds.
