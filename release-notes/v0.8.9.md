# What's new in version 0.8.9

- Added a Playback setting to enable or disable the “Playback finished” screen-reader announcement.
- Kept the end-of-playback state internally even when that announcement is disabled, so Space can still restart from the end correctly.
- Migrated existing old default equalizer shortcuts from `G` to `F4`.
- Rechecked player hotkeys so repeat (`R`), shuffle (`Shift+S`), bass boost (`F3`), and equalizer (`F4`) are all present in Keyboard shortcuts and toggle cleanly.
