# ApricotPlayer 1.0.0 Beta 10

This beta removes the possible startup delay introduced by the beta 7 audio gate.

## Fixes
- Removed the always-on startup mute/pause gate from beta 7.
- mpv now starts immediately again with the target volume, volume-max, speed, ReplayGain, output device, and initial EQ filter already included in the command line.
- Start paused still works: Apricot only starts mpv paused when that user setting is enabled.
- Kept the beta 9 old-player handoff protection, which is no-wait and only affects the player being stopped.

## Notes
- This intentionally prioritizes Apricot's fast startup feel. The old beta 7 gate was conservative, but it could add a small delay while waiting for mpv IPC before unpausing.
- Local installs are not updated automatically from this build unless you choose the beta channel/update manually.
