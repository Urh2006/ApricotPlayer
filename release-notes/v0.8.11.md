# What's new in version 0.8.11

- Fixed manual `cookies.txt` playback retry so a selected cookies file is used after a YouTube login/bot error even when age-restricted fallback support is off.
- Improved Chrome-family cookie export by trying an additional DevTools mode after DPAPI decrypt failures.
- Added diagnostics for selected cookies files, including a warning when no YouTube login cookies are found.
- Added Reset all settings in General and Reset settings for the current section in every Settings section.
- Made settings saves atomic and backed up the previous settings file to reduce update/forced-close resets.
- Throttled repeated automatic cookie-refresh speech after a failed cookie refresh.
- Made Repeat take priority over autoplay/queue when playback reaches the end.
