## What's new?

- the updater now uses a stronger PowerShell replacement script that waits for the old app to fully exit, then replaces the `.exe` and restarts ApricotPlayer
- the updater now checks that the downloaded update is a real Windows executable before installing it
- pressing Escape in player mode now returns focus directly to the search results list
- version numbers now move forward as `0.1.1`, `0.1.2`, `0.2.0`, and eventually `1.0.0` instead of endless beta suffixes
