# What's new in version 0.9

- Chapters are now available from the player. Open the Chapters button or use the Chapters shortcut, press Enter on a chapter to jump there, and press Escape to return to the player.
- Alt+Left and Alt+Right now jump to the previous or next chapter and announce the chapter name for screen reader users.
- Lyrics are now available from the player in a read-only text view. ApricotPlayer first checks for local sidecar lyrics, then can fetch online lyrics from LRCLIB when enabled.
- YouTube comments are now available from the player. With a YouTube Data API key, ApricotPlayer loads comments 20 at a time; without a key it tries a yt-dlp fallback for the first page.
- Keyboard shortcut settings now include chapters, lyrics, comments, previous chapter, and next chapter.
- Playback settings now include mpv-level gapless audio where supported and ReplayGain / loudness normalization.
