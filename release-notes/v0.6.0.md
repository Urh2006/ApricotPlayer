# What's new in version 0.6.0

- Added a Podcasts and RSS feeds screen.
- You can add podcast/RSS feed URLs, refresh all feeds, refresh one feed, open a feed, copy its URL, or remove it.
- Feed item lists support playing an episode, downloading episode audio, copying the item URL, and opening the episode page in the browser.
- RSS and Atom parsing is built in and supports podcast enclosures, Atom enclosure links, publication dates, descriptions, and durations when available.
- When a podcast episode is playing, Escape returns to the feed item list instead of the main menu.
- RSS feed data is stored per user in `%APPDATA%\ApricotPlayer\rss_feeds.json`.

## Downloads

- `ApricotPlayerSetup.exe` for installed Windows builds.
- `ApricotPlayer.zip` for portable builds.
