# ApricotPlayer 0.9.27

## New Features and Improvements

- **Podcast Category / Genre Browsing**: Added a new "Browse categories" button to the RSS Feeds pane. Users can browse top podcasts in 10 core iTunes genres: Arts, Business, Comedy, Education, Music, News, Science, Sports, Technology, and True Crime. Double-clicking or hitting Enter on a category performs a robust iTunes RSS fetch, extracts IDs (with attributes and regex URL fallbacks), performs a batch details lookup, sorts them by original ranking, and displays them within the standard search subscription UI. Back key navigation now also gracefully returns the user from the categories list back to the RSS Feeds screen.
- **SoundCloud Search Provider**: Added a "Search Provider" selection box to the Search pane to toggle between YouTube and SoundCloud. Choosing SoundCloud automatically disables YouTube-specific search type selections and routes search queries through yt-dlp using `scsearch`.
- **Local Media Context Menu Filtering**: Cleaned up the player, playlist, favorites, and history context menus to omit YouTube-specific choices (downloads, stream URLs, channel actions, comments pane, etc.) when the active or selected item is local media. Local media tracks now show "Copy Path" instead of "Copy Link".
- **Menu Focus Preservation**: Returning from any deep-linked submenu to the main menu now preserves the keyboard focus and list item selection on the previous item.

*Note: All core functionality has been verified for stability. Entry-point script (wx_main.py) is kept under 1,000 lines, ensuring clean modularity and stable operation.*
