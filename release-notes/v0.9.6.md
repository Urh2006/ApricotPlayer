# ApricotPlayer 0.9.6

- Fixed folder Previous and Next shortcuts, including the first item in a folder correctly announcing when there is no previous item.
- Stopped folder playback from overwriting the real playback queue.
- Fixed autoplay so queued items are not consumed when "Automatically play next item" is off.
- Added Clear queue to the Playback queue dialog and context menu.
- Fixed player shortcuts across the player screen, including Up/Down for volume and T for time when focus is on the embedded results list.
- Clarified Gapless playback: it only controls mpv's audio transition behavior and does not start the next item.
