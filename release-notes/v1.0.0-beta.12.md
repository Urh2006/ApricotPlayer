# ApricotPlayer 1.0.0 Beta 12

This beta focuses on shortcut discoverability and keeps the YouTube/mpv startup path untouched.

## New
- Main menu actions now show their configured keyboard shortcuts by default, for example Search YouTube shows `Ctrl+Alt+Y`.
- Added a configurable Play file shortcut: `Ctrl+Alt+I`.
- Player buttons now show their configured shortcuts too, except the Play/Pause button, where the plain label stays cleaner for screen readers.
- Added a Settings option to hide shortcut text next to menu and player labels. It is enabled by default.

## Changed
- Removed Action finder from the main menu. It remains available globally with `Ctrl+Shift+J`.
- Action Finder and context menu shortcut labels now follow the same show/hide setting.

## Notes
- This release does not change player startup, stream resolving, mpv arguments, volume, bass boost, or equalizer processing.
- Stable users remain on the stable channel unless they intentionally switch to beta updates.
