# What's new in version 0.9.11

- Fixed next/previous playback choosing from the wrong result collection after dynamic loading.
- Kept the embedded Results selection synchronized with the video that actually starts playing.
- Prevented dynamic next from falling back to shuffle or stale indexes when the exact current URL cannot be continued.
- Blocked player-only shortcuts such as chapters, lyrics, comments, and details while focus is in Results.
- Reset shuffle when opening a normal result directly, so old shuffle state does not make later searches play random videos.
